# Como Contribuir

Obrigado por considerar contribuir para o **ApoioMicro**!

## Fluxo de Contribuição

1. Faça um Fork do projeto.
2. Crie uma branch para a sua _feature_ ou correção (`git checkout -b feature/minha-nova-feature`).
3. Faça o commit das suas alterações (`git commit -m 'feat: adiciona nova funcionalidade'`).
4. Faça o push para a branch (`git push origin feature/minha-nova-feature`).
5. Abra um Pull Request detalhando o que foi feito.

## Regras

- Certifique-se de que todos os testes estão passando (`npm test`).
- Verifique se o código segue os padrões do linter (`npm run lint`).
