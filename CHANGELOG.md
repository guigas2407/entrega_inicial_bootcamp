# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-04-07

### Adicionado

- Lógica de cálculo de precificação baseada em Markup.
- Interface interativa de Linha de Comando (CLI) utilizando Inquirer.js.
- Suíte de testes automatizados com Jest (caminho feliz, entradas inválidas e casos limites).
- Configuração de Linting com ESLint.
- Workflow de Integração Contínua (CI) com GitHub Actions.
- Documentação inicial (README, LICENSE, CONTRIBUTING).
